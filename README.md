# Syn County Sheriff's Office Contraband Registration System

A professional system for registering confiscated contraband for the Syn County Sheriff's Office.

## Features

*   **Categorized Contraband**: Items are organized into Class A, B, C, and D for easy selection.
*   **Weapon Confiscation Policy**: Mandatory popup reminder enforcing SOPs for weapon removal during robberies.
*   **Official Ordinances**: Quick access to the Official Syn County Ordinances.
*   **Briefing Notes**: Integrated briefing notes generator with standard templates.
*   **Automatic Calculations**: Fines are automatically calculated based on the selected items and their quantities.
*   **MDT Ready**: "Copy for MDT" button formats the selected items list specifically for the Mobile Data Terminal.
*   **Privacy Focused**: The system runs entirely in the browser. No data is stored, transmitted, or saved to any server.

## Usage

1.  Click **Open Ledger** to enter the system.
2.  Browse the categories and click on contraband items to add them to the list. (**Note**: A mandatory SOP reminder will appear the first time you select a weapon).
3.  Use the quantity controls (+/- or text input) to adjust the amount of each item.
4.  The fine total is not displayed directly as it is calculated per item class, but the summary provides the breakdown.
5.  Click **Copy for MDT** to copy the formatted list to your clipboard.
6.  Paste the list into the relevant MDT report.
